- Do not ask UI placement questions or low-level technical implementation questions.

## Feedback Interpretation Defaults
- "This feels cluttered" means simplify and reduce competing elements.
- "This is confusing" means improve flow signaling and action clarity.
- "This feels boring" means increase pacing, stakes, and feedback texture.
- "This is not intuitive" means redesign interaction flow autonomously.
